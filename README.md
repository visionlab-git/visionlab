# VisionLab - Studio de Traitement d'Images Avancé 🚀

**VisionLab** est une plateforme web interactive de traitement d'images conçue pour l'excellence académique à l'Université de Yaoundé 1. Elle permet d'explorer les concepts fondamentaux et avancés de la vision par ordinateur à travers une interface moderne et intuitive.

![VisionLab Logo](/logo.png)

## 🌟 Fonctionnalités

Le studio est organisé en 6 chapitres thématiques :

1.  **Traitement de base** : Luminosité, contraste, égalisation d'histogramme, opérations arithmétiques et interpolation (Voisin, Bilinéaire).
2.  **Convolution** : Filtrage spatial avec masques (Moyenneur, Gaussien, Passe-haut, Médian).
3.  **Analyse de Fourier** : Passage dans le domaine fréquentiel via FFT, visualisation du spectre et filtrage idéal (LPF/HPF).
4.  **Détection de Contours** : Algorithmes de Sobel, Prewitt, Robert, Laplacien, Gradient et Isotrope.
5.  **Segmentation** : Seuillage manuel et automatique via la méthode d'Otsu.
6.  **Morphologie Mathématique** : Érosion, Dilatation, Ouverture, Fermeture et Gradients morphologiques.

## 🛠️ Stack Technique

-   **Frontend** : React 19 + Vite + Tailwind CSS
-   **Animations** : Lucide React + Tailwind Transitions
-   **Algorithmes** : JavaScript natif optimisé (Float32Arrays, Typed Arrays)
-   **Visualisation** : Canvas 2D API avec optimisation `willReadFrequently`
